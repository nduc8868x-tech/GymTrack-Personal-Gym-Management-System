## Project Overview

**GymTrack** is a personal web application that helps users self-manage and optimize their gym training process, including scheduling, workout tracking, nutrition logging, and AI-powered coaching.

**Problem Statement:**
Gym-goers typically lack a unified tool to track workouts, nutrition, and progress all in one place. Existing apps are either too complex (designed for gym management businesses) or lack personalization. GymTrack solves this by combining workout tracking, nutrition logging, and AI coaching in a single, streamlined application focused on the individual user.

| Attribute | Value |
|-----------|-------|
| **Platform** | Web App (Production) |
| **Type** | Single-user personal fitness tracker |
| **Tech Stack** | Next.js (FE) + Node.js/Express (BE) + PostgreSQL (DB) |
